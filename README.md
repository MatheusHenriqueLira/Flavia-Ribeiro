## 📌 Sobre o Projeto

Este projeto está sendo desenvolvido como parte de um trabalho acadêmico. O objetivo principal é construir uma interface web moderna, organizada e responsiva utilizando as melhores práticas de desenvolvimento front-end.

## 🛠️ Tecnologias Utilizadas

O projeto foi estruturado utilizando as seguintes tecnologias:

* **HTML5** - Estruturação de conteúdo e páginas.
* **CSS3** - Estilização, layout (Flexbox) e design responsivo.
* **JavaScript** - Engenharia de interatividade e lógica do usuário.

## 📁 Estrutura de Pastas

A organização dos arquivos no repositório segue o padrão abaixo:

* `html/` - Páginas e estruturas do site (incluindo o `index.html`).
* `css/` - Arquivos de estilização (`style.css`).
* `js/` - Scripts e comportamentos em JavaScript.
* `img/` - Banco de imagens e mídias utilizadas no layout.
